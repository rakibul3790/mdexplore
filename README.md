## 📋 What is mdexplore?

mdexplore is a desktop tool designed to help you open and view Markdown files quickly on Windows. It acts like a file explorer, but focused on displaying `.md` files with added features. You can highlight text, copy in bulk, or open files for editing in VS Code. It supports heavy Markdown browsing tasks, which makes it useful for managing big collections of documents.

This program runs on Python and uses PySide6 to create its interface. It includes support for diagrams like Mermaid and PlantUML, and renders math formulas clearly with MathJax.

## 📂 Using mdexplore

### Browsing Files

Use the folder explorer on the left side to navigate through your documents directory. mdexplore shows files with the `.md` extension.

### Viewing Markdown Files

Click any Markdown file to see its content instantly. The app transforms the plain text into a nicely formatted preview, including support for:

- Headings, lists, and tables
- Code blocks
- Images linked from your folders
- Mermaid diagrams and PlantUML diagrams
- Math formulas rendered via MathJax

### Highlighting and Copy

Select text directly in the preview window. You can:

- Highlight important sections.
- Copy large sections of text easily to paste elsewhere.
- Use keyboard shortcuts like Ctrl+C for copying.

### Editing with VS Code

If you want to edit a file, right-click it and choose **Open in VS Code**. This action uses your installed Visual Studio Code to open the file for detailed editing.

## ⚙️ Configuration and Settings

mdexplore lets you adjust viewing options:

- Change font size for better readability.
- Toggle dark mode for low-light environments.
- Enable or disable diagram rendering if you want simpler, faster previews.
- Set default folder to open on startup.

You can find these options under the **Settings** menu in the app’s main window.

## 🛠️ Troubleshooting Tips

- If Markdown files do not show up, make sure you are opening a folder that contains `.md` files.
- If diagrams don’t render, check settings to confirm diagram support is enabled.
- For editing, ensure Visual Studio Code is installed and set correctly on your system.
- If the app crashes or won’t open, try reinstalling from the latest download page linked above.
